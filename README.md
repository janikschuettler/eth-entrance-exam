# ETH Entrance Exam Resources

This repository serves as the backend for the website at [janiks.me](https://janiks.me/), a comprehensive collection of ETH Zurich entrance exam materials including past written exams, oral exam protocols, and preparation tips. 

## Contents

### Written Exams
- **Past exams (2001-2024)**: Mathematics I & II, Physics
- **Official ETH examples**: Sample questions from the official ETH website
- **Practice materials**: Including the new exam format introduced in 2021

### Oral Exams
- **Exam protocols (2017-2026)**: Memory protocols from former exam candidates
- **Exam types**: Both reduced and comprehensive oral exam formats
- **Multiple perspectives**: Protocols from various candidates showing different exam experiences

Disclaimer: The oral exam protocols are memory protocols from former candidates and are not official ETH Zurich documents. They should serve as orientation for the scope and depth of the examined material. Oral exams are individually developed and may vary for different candidates.
